# Статусы (пайплайн) работы с пациентом

Ниже — пример статусов для ведения кейсов медицинского координатора (второе мнение / лечение за рубежом). Статусы можно использовать в Notion/Trello/Jira/GitHub Projects.

## Воронка / этапы
1. **Лид (новый запрос)**
   - есть первичное сообщение/заявка, данных мало

2. **Скрининг / уточнение**
   - собраны базовые сведения: диагноз/подозрение, срочность, страна/язык, ожидания
   - принято решение: второй взгляд / пересмотр снимков / пересмотр гистологии / организация поездки

3. **Офер / согласование пакета**
   - согласован состав работ, сроки, стоимость вашей координации
   - выставлен счёт/ссылка на оплату, отправлены условия сервиса

4. **Оплата получена**
   - оплачена ваша работа (координация/упаковка кейса). Оплаты клинике/врачу — отдельно, по правилам выбранного провайдера

5. **Сбор документов**
   - получены выписки/анализы/заключения
   - получены файлы DICOM (КТ/МРТ) при необходимости

6. **Перевод / подготовка кейса**
   - выполнен перевод ключевых документов
   - подготовлен **Medical case summary** (хронология + список вопросов)

7. **Подбор клиники/врача**
   - подобраны 1–3 варианта
   - согласованы сроки ответа и стоимость консультации

8. **Запись / подтверждение консультации**
   - консультация назначена, подтверждены время, формат, требования

9. **Консультация проведена**
   - получено заключение/письмо врача/результат консилиума

10. **Выдача результата клиенту**
   - передано заключение
   - проведён разбор «человеческим языком»
   - сформированы следующие шаги (next steps)

11. **Координация лечения (опционально)**
   - согласование плана лечения, графика процедур
   - смета/счета, организационные вопросы

12. **Поездка / лечение (опционально)**
   - визовая поддержка (без гарантий)
   - логистика, сопровождение, переводчик

13. **Пост‑сопровождение**
   - контрольные анализы/наблюдение
   - коммуникация с клиникой по результатам

14. **Закрыто (успешно)**
   - кейс завершён, собрана обратная связь

15. **Закрыто (неуспешно/отказ)**
   - клиент передумал/не смог оплатить/нет контакта/не подходит по критериям

## Служебные статусы
- **Ожидаю клиента** — жду документы/ответ/оплату
- **Ожидаю партнёра** — жду переводчика/клинику/врача
- **Стоп: юридические/этические риски** — запрос на «серые» действия, подделки, гарантии результата и т.п.
