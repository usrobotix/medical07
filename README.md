# medical07

Проект **medical07** — заготовка под сервис/проект по роли **медицинского координатора** (patient concierge): организация "второго мнения" и (опционально) сопровождение лечения за рубежом.

## Цель
Собрать в одном месте:
- описание модели услуг (что делаем/что не делаем)
- статусы работы с пациентом (воронка/пайплайн)
- текстовый разбор направления (razbor.md)

## Структура репозитория
- `README.md` — описание проекта
- `status.md` — статусы/этапы работы (пайплайн)
- `razbor.md` — подробный разбор: услуги, процесс, риски, ценообразование

## Важно
Этот репозозиторий **не является медицинской консультацией**. Медицинские решения принимает только врач. Здесь описывается организационная/координационная часть сервиса.
