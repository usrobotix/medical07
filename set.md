## Три слоя партнёрской сети

```
┌────────────────────────────────────────────────────────────────┐
│  Слой 1: Клиники / International Patient Office (IPO)          │
│  — принимают пациента, ставят диагноз, оказывают лечение       │
├────────────────────────────────────────────────────────────────┤
│  Слой 2: Переводчики с медицинской специализацией              │
│  — письменный перевод документов; устный на приёме (онлайн)    │
├────────────────────────────────────────────────────────────────┤
│  Слой 3: Медицинский редактор / куратор кейса                  │
│  — врач-консультант, который помогает структурировать вопросы  │
│    и проверить правильность сборки кейса (без постановки       │
│    диагноза от имени вас)                                       │
└────────────────────────────────────────────────────────────────┘
```

## Чек-лист верификации партнёра

### Для клиники / IPO

- [ ] Лицензия учреждения проверена на сайте регулятора страны (или реестр аккредитации JCI/ISO).
- [ ] IPO-контакт является сотрудником клиники (не посредником).
- [ ] Счета выставляются официально от юрлица клиники (не наличными и не «на карту»).
- [ ] Пройден тестовый кейс: ответ получен в согласованный срок, заключение содержательное.
- [ ] Согласован SLA: первичный ответ ≤ 48 ч, заключение ≤ 10 рабочих дней.
- [ ] Получены условия работы с координаторами (комиссионная политика или её отсутствие).

### Для переводчика

- [ ] Есть примеры медицинских переводов (аналогичная ниша).
- [ ] Тестовый фрагмент (1–2 страницы) переведён без грубых ошибок в терминологии.
- [ ] SLA зафиксирован письменно (мессенджер/email).
- [ ] Подписано или согласовано NDA (или согласие на конфиденциальность в простой форме).
- [ ] Стоимость зафиксирована с формулой (страница/символ/час).

### Для медицинского редактора / куратора

- [ ] Специализация соответствует нише.
- [ ] Нет конфликта интересов (не аффилирован с клиниками из вашей сети без раскрытия).
- [ ] Чётко понимает свою роль: структурирование, а не постановка диагноза пациенту напрямую.
- [ ] Готов работать в согласованном SLA (≤ 24–48 ч на review кейса).
- [ ] Подписано NDA.

## Минимальная стартовая сеть

Для запуска достаточно следующего минимума:

| Слой | Минимум | Цель |
|---|---|---|
| Клиники | 2–3 клиники в 2 странах (разные профили) | Покрыть 1–2 целевые ниши |
| Переводчики | 1 основной (EN↔RU) + 1 резервный | Не зависеть от одного |
| Медкуратор | 1 врач-консультант по целевой нише | Качество сборки кейса |

**Приоритет стран для старта**: Турция или Израиль — быстрый ответ IPO, опыт работы с русскоязычными пациентами, понятное ценообразование.

## Шаблоны писем и сообщений

### Email клинике / IPO (на английском)

```
Subject: Cooperation Inquiry — Medical Coordinator for Russian-Speaking Patients

Dear International Patient Office team,

My name is [Your Name]. I am a medical coordinator based in [City/Country],
specializing in second opinion and treatment coordination for Russian-speaking
patients.

I am reaching out to explore the possibility of establishing a cooperation
agreement with [Clinic Name]. Specifically, I am interested in:

1. Understanding your process for international patient referrals.
2. Discussing case submission formats (medical summaries, DICOM, pathology).
3. Learning about your standard response SLA for second opinion requests.
4. Understanding your invoicing process for international patients.

I currently work with patients primarily in the fields of [oncology / orthopedics /
cardiology — choose relevant]. My role is strictly organizational: I prepare and
translate patient cases, coordinate scheduling, and support patients throughout
the process. I do not provide medical advice.

I would appreciate a 20–30 minute call or a brief overview of your international
referral protocol.

Best regards,
[Your Name]
[Phone / Telegram / Email]
[Website / LinkedIn, if available]
```

### Письмо медицинскому редактору / куратору (на русском)

```
Тема: Сотрудничество — куратор медицинских кейсов (второе мнение)

Добрый день, [Имя Отчество]!

Меня зовут [Ваше имя], занимаюсь координацией получения второго мнения
для пациентов в зарубежных клиниках.

Ищу врача-консультанта по [специальность] для следующей роли:
- проверка правильности сборки кейса (хронология, список вопросов)
- оценка полноты документов перед отправкой в клинику
- разбор заключения с пациентом «человеческим языком» после консультации

Важно: вы не ставите диагноз пациенту — только помогаете мне как координатору
структурировать кейс и правильно интерпретировать ответ клиники.

Формат сотрудничества: разовые кейсы, оплата за кейс-ревью.
NDA, конфиденциальность — само собой.

Если интересно — давайте созвонимся на 20 минут?

С уважением,
[Имя], [контакты]
```

## 7-дневный план действий без бюджета

| День | Задачи |
|---|---|
| **День 1** | Определить целевую нишу (1 штука). Составить список из 5–7 клиник в 1–2 странах по нише. Найти контакты IPO на сайтах клиник. |
| **День 2** | Отправить первые 3–5 email/WhatsApp в IPO по шаблону выше. Одновременно: найти 3–5 медпереводчиков (hh.ru, LinkedIn, ProZ, telegram-каналы переводчиков). |
| **День 3** | Написать 2–3 переводчикам по шаблону. Найти 1–2 врача-куратора по нише (коллеги, LinkedIn, профессиональные чаты). |
| **День 4** | Написать врачу-куратору по шаблону. Подготовить базовый чек-лист документов для целевой ниши (1 страница). |
| **День 5** | Обработать первые ответы от IPO/переводчиков. Провести тестовый созвон с 1 переводчиком и/или куратором. |
| **День 6** | Заполнить карточки партнёров по шаблонам выше. Провести верификацию по чек-листу для тех, кто ответил. |
| **День 7** | Итог: у вас есть минимальная сеть из 1–2 клиник, 1 переводчика, 1 куратора. Готовы принять первый кейс. Опубликовать первый пост/объявление о поиске первого клиента. |

> **Ключевое правило**: не ждать "идеальной" сети. 1 клиника + 1 переводчик + 1 куратор = достаточно для старта.
