# Маркетинг‑план на 60 дней (MVP, средний чек)

## 1. Офферы
### Основной
- «Второе мнение (онлайн)» — упаковка кейса + подбор клиники/врача + организация консультации.

### Лид‑магнит
- Бесплатный чек‑лист документов + 10–15 минутный вводный созвон.

### Апселл
- «Организация визита/лечения» — координация поездки и коммуникации.

## 2. Каналы привлечения
1) Telegram: тематические чаты (по запросу, без спама)
2) Партнёрства в РФ: МРТ/КТ центры, лаборатории, реабилитация
3) Рефералы от врачей (РФ/СНГ): сложные случаи на second opinion
4) Контент (SEO): гайды по документам, DICOM, гистология, «как получить второе мнение»
5) Позже: таргет/контекст на лид‑магнит

## 3. Воронка
Контент/чат → чек‑лист → короткий созвон → оплата упаковки кейса → second opinion → апселл.

## 4. Процессы
- Скрипт первичного созвона
- SLA ответов клиенту (например 2 часа в рабочее время)
- Таблица «источник лида» в CRM

## 5. KPI
- Конверсия лид → созвон
- Конверсия созвон → оплата
- Среднее время до получения заключения
- Повторные обращения/рефералы
