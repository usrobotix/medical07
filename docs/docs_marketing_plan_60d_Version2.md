# Маркетинг‑план на 60 дней (MVP, средний чек)

## 1) Офферы
- Основной: «Второе мнение (онлайн)» — упаковка кейса + подбор клиники/врача + организация консультации.
- Лид‑магнит: чек‑лист документов + 10–15 минутный вводный созвон.
- Апселл: «Организация визита/лечения».

## 2) Каналы лидов
1) Telegram‑чаты по запросу (не спам) + собственный канал
2) Партнёрства в РФ: МРТ/КТ, лаборатории, реабилитация
3) Рефералы от врачей РФ/СНГ (сложные случаи на second opinion)
4) Контент/SEO: гайды по DICOM/гистологии/документам/странам
5) Позже: платная реклама на лид‑магнит

## 3) Воронка
Контент → чек‑лист → созвон → оплата упаковки кейса → second opinion → апселл.

## 4) KPI
- Лид → созвон
- Созвон → оплата
- Среднее время ответа клиники
- Доля рекомендаций/повторных
